# PHP-Proxy-Docker


## Dockerfile (Build the image yourself)

1. clone the repo
2. docker build -t php-proxy .
Running 
1. docker run -i -t php-proxy


## docker-compose

1. clone the repo
2. docker-compose up (-d for background)
